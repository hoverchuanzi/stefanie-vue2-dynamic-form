### 动态表单实现功能

-[x] 可以通过字段配置渲染表单。
-[ ] 支持至少 6 种基础控件。
-[ ] 支持新增、编辑、详情三种模式。
-[ ] 支持必填、正则、数字等基础校验。
-[ ] 支持字段显隐、禁用和至少 2 个字段联动。
-[ ] 支持提交前数据格式化。 
-[ ] 至少有 3 个业务表单示例。
-[ ] 可以用 2-3 分钟讲清楚实现思路。
